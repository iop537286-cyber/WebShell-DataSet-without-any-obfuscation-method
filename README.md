Overview

This dataset contains a collection of baseline PHP WebShell samples used as the original input set for subsequent layered transformation and evaluation experiments. All samples in this dataset are raw, unobfuscated WebShells, meaning that they preserve their original command-receiving logic and execution behavior without any lexical, control-flow, data-flow, or other obfuscation applied.

The purpose of this dataset is to provide a unified starting point for studying how large language models can be used to generate transformed WebShell variants under controlled experimental settings.

Dataset Characteristics

Language: PHP

Sample type: Raw WebShell samples

Obfuscation status: None

Role in the workflow: Baseline input dataset for layered sample generation

Intended use: Controlled research on code transformation, layered rewriting, and sample evolution

Each sample is designed to preserve its original behavior while exposing a clearly identifiable command input path and command execution path. Therefore, the dataset serves as the initial, non-obfuscated reference set before any prompt-driven transformation is performed.

Input Entry Categories

The dataset is organized according to different command entry channels, i.e., different ways in which command content is received from the request context. The current dataset includes the following entry categories:

BODY_POST
Samples in this category receive command data from the HTTP request body via POST-related mechanisms.

COOKIE
Samples in this category read command content from cookie fields.

FILE_UPLOAD
Samples in this category obtain command-related content from uploaded file metadata or file-related request structures.

HEADER
Samples in this category read command content from HTTP header fields.

META
Samples in this category use metadata-related request carriers or auxiliary request-associated fields as the command entry source.

SESSION_META
Samples in this category use session-associated metadata as the command entry channel.

URL_QUERY
Samples in this category receive command content through URL query parameters.

These categories reflect different request-side entry mechanisms and are used to ensure diversity at the input-source level before later transformations are applied.

Command Execution Behavior

All samples in this dataset are baseline command-execution WebShells. In other words, each sample contains:

a command entry point, through which external command content is obtained; and

a command execution point, through which the received command is passed to the underlying execution logic.

At this stage, the dataset does not introduce any additional concealment, structural perturbation, or transformation strategy. The command reception and execution logic remain in their direct and original form, making the dataset suitable as a reference baseline for subsequent multi-layer rewriting experiments.

Directory Organization

The dataset is grouped by command entry category. Each top-level directory corresponds to one command entry type, for example:

BODY_POST/

COOKIE/

FILE_UPLOAD/

HEADER/

META/

SESSION_META/

URL_QUERY/

This organization facilitates category-based sampling, batch processing, and layered transformation in later experimental stages.

Experimental Role

In the overall research workflow, this dataset functions as the starting sample pool. Later transformed samples are generated by applying structured prompt-driven rewriting strategies to these baseline files. Therefore, this dataset should be understood as:

the original sample source,

the non-obfuscated baseline set, and

the reference input for layered transformation pipelines.

Notes

All samples are baseline samples without any obfuscation.

The dataset is intended to support controlled experimental research on code transformation and layered sample generation.

The folder structure reflects different command entry channels, not different obfuscation methods.

Any transformed or layered variants derived later should be treated as separate datasets or derived outputs rather than part of this baseline set.
